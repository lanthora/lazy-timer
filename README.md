# Lazy Timer
Python实现的一个很懒的计时器。

这个计时器对CPU没有兴趣，总想着睡觉。

所以它工作起来总是会有误差，误差大小取决与它深睡的程度。

虽然从设计上来说它最多睡一分钟，但是没有经过太多测试谁也不好说它不会不多睡一会。
